---
category: data
heading: data(object)
---

The `.data(object)` method will set multiple key/value pairs (overwriting if already set) on the collection if an object is passed in.

    var rect = Pablo.rect();
    
    rect.data({
      foo: 'bar',
      biz: {
        yin: 1,
        yan: 2
      }
    });
    
    alert(rect.data('foo'));
    alert(rect.data('biz'));
